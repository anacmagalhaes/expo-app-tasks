# Expo App Tasks

## Descrição
Aplicação mobile de gerenciamento de tarefas desenvolvida com React Native e Expo. Permite criar, listar, concluir e remover tarefas, com persistência local utilizando SQLite.

---

## Arquitetura e Tecnologias
O projeto é uma aplicação mobile standalone, focada em gerenciamento local de dados:

- Frontend (React Native + Expo): Interface mobile para criação e gerenciamento de tarefas.
- Persistência de Dados (SQLite): Armazenamento local das tarefas no dispositivo.
- Gerenciamento de Estado: Controle das tarefas e atualização da interface em tempo real.

---

## Funcionalidades
• Criação de tarefas
• Listagem de tarefas
• Marcar como concluída
• Remoção de tarefas
• Persistência local dos dados

---

## Fluxo de Funcionamento
- O usuário cria uma nova tarefa pelo app
- A tarefa é armazenada localmente no SQLite
- A lista é atualizada automaticamente na interface
- O usuário pode concluir ou remover tarefas a qualquer momento
